#  Hotel Booking Platform Testing

Manual functional testing of the core interaction forms to ensure input validation and system stability.

##  Quick Links
* **Tested Website:** [Shady Meadows B&B](https://automationintesting.online/)
* **Bug Reports:** Tracked directly in the [Issues](https://github.com/nprok/QA_portfolio-/issues) tab.

##  Testing Scope (What I Checked)
* **Positive Paths:** Submitting fields with valid data formats and separators.
* **Boundary & Layout:** Testing text fields for character limits (5 to 2000 chars) and layout breaks.
* **Negative Validation:** Verification of blocked empty submissions, invalid email formats (missing TLD), and restricting letters in the phone input field.
